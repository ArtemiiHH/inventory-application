# Sneaker Store Website

[Live Demo](https://example.com)

A full-stack sneaker store inventory app built with Node.js, Express, and PostgreSQL.
Tech Stack

- Backend: Node.js, Express REST API
- Database: PostgreSQL
- Templating: EJS
- File Uploads: Multer

# Setup

1. Install dependencies: `npm install`
2. Configure your database in `db/pool.js`
3. Add categories at `/categories/new`
4. Populate the DB: `node db/populatedb.js`
5. Start the server: `node app.js`
