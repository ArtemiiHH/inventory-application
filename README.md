#Sneaker Store 👟
🔗 Live Demo
A full-stack sneaker store inventory app built with Node.js, Express, and PostgreSQL.
Tech Stack

Backend: Node.js, Express REST API
Database: PostgreSQL
Templating: EJS
File Uploads: Multer

#Setup

Install dependencies: npm install
Configure your database in db/pool.js
Add categories at /categories/new
Populate the DB: node db/populatedb.js
Start the server: node app.js
